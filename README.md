# Coco Restaurant Finder Bot

## Descrription

This project was made to test a simple Conversational Information Retrieval Bot to find restaurants that may be relevant for the user. The algorithm used to calculate the ranking of the results was Weight Terms and the implementation was made by using the Google Search API.

## How to run it?

First, install the package needed for the Telegram bot:

```
pip install python-telegram-bot -U --pre 
```

And then, just run the command: ``` py -3 newbot.py```
